# 🏛️ Maison Matrix - Architectural Studio Website

A premium, modern full-stack web application designed for a luxury Architectural Studio. This project showcases minimalist design, responsive structures, interactive cinematic frame zones, and a secure contact/inquiry backend connected to a database.

## ✨ Features

*   **Premium Visuals:** Immersive Hero Section with custom image scales and gradients to give a cinematic studio preview.
*   **Sleek Navigation:** A beautiful, responsive capsule-shaped navbar that automatically fits all screen form-factors.
*   **Cinematic Frame Zone:** A curated layout that fluidly resizes typography on mobile screens for seamless user experience.
*   **Secure Backend Architecture:** Contact and inquiry forms managed via a secure Node.js & Express server.


---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6)
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB Atlas (Mongoose ODM)
*   **Environment Management:** Dotenv
*   **Version Control:** Git & GitHub
